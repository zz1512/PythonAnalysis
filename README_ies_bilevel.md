# 氢氨醇一体化综合能源系统双层规划示例

该示例实现了一个面向**零净排放**的双层优化框架：

- **上层**：容量配置（风电、光伏、电解槽、储氢、合成氨、合成甲醇、CCUS、电池）
- **下层**：给定容量下的逐时能源调度（Gurobi 线性规划）

## 安装依赖

```bash
pip install gurobipy
```

并确保已安装 Gurobi 且许可证可用。

## 运行

```bash
python ies_bilevel_planning.py --hours 24
python ies_bilevel_planning.py --hours 24 --compare --sensitivity
```

## 模型说明

- 下层目标最小化运行成本：购电成本 + 燃气成本 + 捕集成本 + 碳成本
- 上层通过离散容量搜索结合下层最优调度，最小化年化总成本
- 零净排放约束：`总排放 - 捕集量 <= 0`

> 注：该示例用于科研建模与方法验证，参数为演示级，建议根据真实项目数据校准。
